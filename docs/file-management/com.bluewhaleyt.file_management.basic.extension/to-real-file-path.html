Deprecated

This function is deprecated because it uses file paths to handle file management, which may cause unexpected behavior or errors on newer versions of Android. The Uri to be strictly converted as an actual file path is not highly recommended. Use it at your own risk. Please use it for just DISPLAYING real file path to users.
